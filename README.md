# v league data api

## 🔨Setup
### Create container
```
docker-compose build
```

### Run container
```
docker-compose up -d
```

### Enter container
```
docker-compose exec python bash
```

## 🏃Run
### Start container
```
doker-compose start
```
```
python3 main.py
```
```
docker-compose stop
```
